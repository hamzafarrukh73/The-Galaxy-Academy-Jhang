# The Galaxy Academy Jhang
A modern, high-performance landing page and management dashboard for The Galaxy Academy, Jhang. Built with a cutting-edge technological stack ensuring speed, reliability, and excellent user experience.

## 🚀 Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (Recommended runtime)
- [Node.js](https://nodejs.org/) v22 or higher

### Tech Stack

- **Core Framework:** [Nuxt 4](https://nuxt.com/) (Vue 3)
- **UI & Styling:** [Nuxt UI](https://ui.nuxt.com/) & [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** [Pinia](https://pinia.vuejs.org/)
- **Backend & Auth:** [Supabase](https://supabase.com/)
- **Runtime & Package Manager:** [Bun](https://bun.sh/)
- **Validation:** [Zod](https://zod.dev/)
- **Icons:** [Iconify](https://iconify.design/) (Lucide & Simple Icons)

## 📁 Project Structure

```bash
app/
  ├── api/          # API client, repositories, types, errors
  ├── assets/       # Global CSS, images, fonts
  ├── components/   # Atomic Vue components
  ├── composables/  # Shared Vue composables
  ├── layouts/      # Layout templates for different routes
  ├── middleware/   # Navigation guards and route logic
  ├── pages/        # File-based routing for application pages
  ├── plugins/      # Nuxt plugins for third-party integrations
  ├── repository/   # Data fetching logic and patterns
  ├── schemas/      # Zod validation schemas
  ├── stores/       # Pinia state management modules
  ├── types/        # TypeScript definitions and database types
  └── utils/        # Shared helper functions
```
