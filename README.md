# The Galaxy Academy Jhang
A landing page for the academy.

## Project Structure

```
app/
  ├── api/          # API client, repositories, types, errors
  ├── components/   # Vue components
  ├── layouts/      # Layout templates
  ├── pages/        # Route pages
  ├── stores/       # Pinia stores
  ├── middleware/   # Route middleware
  ├── plugins/      # Nuxt plugins
  └── utils/        # Utilities
```
